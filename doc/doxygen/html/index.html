Bookify Developer Manual

  1. Requirements

        All local .h files will be compiled when the program is run

        The tests depend on an external library 'Check' which can be installed using the

        instructions for your machine here: https://libcheck.github.io/check/web/install.html

  1. Getting started

        Compile the program from the root folder by calling make.

        You can also call [make bookify] and make [all_tests] separately.

        

        The executable will be located in the bin folder.

  1. Tests

        Run tests by launching all_tests from the bin folder

        Test output will be in a file called tests.log in the root directory.
